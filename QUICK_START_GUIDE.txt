WHAT IS THIS APPLICATION?
--------------------------
A desktop-first cash book management system for tracking receipts and payments
with features like:
- Easy Receipt/Payment entry
- Automatic date filling
- Autocomplete for previous entries
- Running balance calculation
- Connected to Nile PostgreSQL Database

================================================================================

HOW TO RUN THE APPLICATION
---------------------------

OPTION 1: Double-Click the BAT File (EASIEST)
----------------------------------------------
1. Locate the file: START_SMP_CASHBOOK.bat
2. Double-click it
3. Wait 10 seconds for both servers to start
4. Browser will open automatically at http://localhost:5173
5. Start using the Cash Book!

OPTION 2: Manual Start
-----------------------
1. Open Command Prompt #1:
   - Navigate to: smp-cashbook-backend
   - Run: npm start

2. Open Command Prompt #2:
   - Navigate to: smp-cashbook
   - Run: npm run dev

3. Open browser to: http://localhost:5173

================================================================================

HOW TO STOP THE APPLICATION
----------------------------

OPTION 1: Using STOP Bat File
------------------------------
1. Double-click: STOP_SMP_CASHBOOK.bat
2. All servers will be stopped automatically

OPTION 2: Manual Stop
---------------------
1. Go to each Command Prompt window
2. Press Ctrl+C
3. Type 'Y' and press Enter
4. Close the windows

HOW TO USE THE CASH BOOK
-------------------------

ADDING A RECEIPT:
1. Click the green "Receipt" button
2. Fill in the form:
   - Date: Auto-filled from last entry (format: dd/mm/yy)
   - Cheque No: Optional
   - Amount: Required (positive number)
   - Head of Accounts: Required (autocomplete available)
   - Notes: Optional
3. Click "Save Receipt"

ADDING A PAYMENT:
1. Click the red "Payment" button
2. Fill in the form (same as receipt)
3. Click "Save Payment"

EDITING AN ENTRY:
1. Find the entry in the table below
2. Click "Edit" button on the right
3. Modify the fields
4. Click "Save" button

DELETING AN ENTRY:
1. Find the entry in the table
2. Click "Delete" button
3. Confirm deletion

FEATURES
--------

✓ Color-Coded Entries:
  - Green rows = Receipts (money in)
  - Red rows = Payments (money out)

✓ Running Balance:
  - Shows balance after each transaction
  - Automatically calculated

✓ Autocomplete:
  - Start typing in any field
  - Previous entries will be suggested
  - Press Arrow keys to navigate suggestions
  - Press Enter to select

✓ Smart Date Entry:
  - Auto-fills from last entry date
  - Format: dd/mm/yy (example: 12/12/24)
  - Validates date automatically

✓ Duplicate Detection:
  - Warns if similar entry was made within 5 seconds
  - Prevents accidental duplicates

✓ Summary Footer:
  - Total Receipts
  - Total Payments
  - Net Balance (Receipts - Payments)

TROUBLESHOOTING
---------------

PROBLEM: Blank white screen
SOLUTION:
- Make sure backend server is running first
- Wait 10 seconds after starting
- Refresh the browser (F5)
- Check if port 3001 and 5173 are not blocked

PROBLEM: "Failed to fetch entries" error
SOLUTION:
- Check if backend server is running
- Look for green checkmark: "Connected to Nile Database"
- Check internet connection (Nile requires internet)

PROBLEM: Port already in use
SOLUTION:
- Run STOP_SMP_CASHBOOK.bat first
- Then run START_SMP_CASHBOOK.bat again
- Or restart your computer

PROBLEM: Cannot connect to database
SOLUTION:
- Check internet connection
- Verify .env file in smp-cashbook-backend folder
- Contact system administrator
