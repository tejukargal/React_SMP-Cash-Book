HOW TO USE THE APPLICATION
---------------------------

QUICK START (RECOMMENDED):
1. Double-click: START_SMP_CASHBOOK.bat
2. Wait 10 seconds
3. Browser opens automatically
4. Start entering receipts/payments

MANUAL START:
1. Terminal 1: cd smp-cashbook-backend && npm start
2. Terminal 2: cd smp-cashbook && npm run dev
3. Open: http://localhost:5173 (or check terminal for port)

TO STOP:
- Double-click: STOP_SMP_CASHBOOK.bat
- Or press Ctrl+C in both terminal windows

SUPPORT & MAINTENANCE

For technical support:

2. Check error messages in terminal windows

4. Restart using START_SMP_CASHBOOK.bat

- Backend code: smp-cashbook-backend/server.js
- Database: Use Nile MCP or pgAdmin

Common Issues:
- Blank screen: Backend not started, wait longer
- Port in use: Run STOP_SMP_CASHBOOK.bat first
- Database error: Check internet connection
- Form not saving: Check browser console (F12)

INSTALLATION ON NEW COMPUTER
-----------------------------

If you need to install on another computer:

1. Copy entire "React_Cash Book" folder
2. Install Node.js (v18 or higher)
3. Open terminal in smp-cashbook-backend:
   npm install
4. Open terminal in smp-cashbook:
   npm install
5. Double-click START_SMP_CASHBOOK.bat

All dependencies will be automatically installed.
Database connection will work from any computer with internet.
